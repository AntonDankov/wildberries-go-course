## Features

**Built-in Commands:**

- `cd <path>` - Change directory
- `pwd` - Print working directory
- `echo <text>` - Display text
- `kill <pid>` - Terminate process
- `ps` - List running processes
- `exit` - Exit shell

**Advanced Features:**

- Pipeline support using `|` operator
- External command execution
- `Ctrl+C` - Interrupt current command
- `Ctrl+D` - Exit shell

## How to Run

Example of running and executing commands

```bash
go run .
mini-shell> pwd
mini-shell> echo Hello World
mini-shell> ps | findstr go
mini-shell>ping google.com
mini-shell> exit

```
